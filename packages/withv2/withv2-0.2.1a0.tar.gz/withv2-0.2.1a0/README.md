CLI to interact with Twitter API v2

  withv2/0.2.1-alpha

  $ withv2 [COMMAND]

  tweets          manage tweet objects, streams 
  users           manage follows, blocks & mutes
  spaces          lookup and search spaces
  lists           manage user's lists
